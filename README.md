# ex_lib
UI

### Steps -

# 1. Clone the repo

```bash
    git clone httpsPATH
```

# 2. Create virtual env

```bash
    conda create -n venvel python=3.8 -y
```

# 3. Activate virtual env

```bash
    conda activate venvel
```

# 4. Install requirements

```bash
    pip install -r requirements.txt
```
